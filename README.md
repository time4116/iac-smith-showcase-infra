## Warnings

* No data transformation or validation is configured; raw records will be written to S3
* No dead-letter queue or failed record handling mechanism is specified; records exceeding Firehose limits may be lost
* S3 bucket lifecycle policies are not configured; versioned objects will accumulate indefinitely
* No monitoring or alarms are configured for stream throughput, delivery latency, or failed deliveries
* Firehose does not natively support schema validation; data quality depends on upstream producers
