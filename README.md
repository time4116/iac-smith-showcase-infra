# iac-smith-showcase-infra

Public showcase target for [IaC Smith](https://github.com/time4116/iac-smith): every pull request in this repository was generated by the controller from a natural-language GitHub issue — parsed into typed intent, composed as schema-validated resource specs, rendered deterministically, and gated on static review plus real `terragrunt plan` checks before the PR was opened.

Nothing here is applied automatically. The generated apply workflow is gated behind a GitHub Environment approval, and IaC Smith itself never runs `terraform apply`.

Browse the issues and their linked pull requests to see the issue → PR pipeline end to end.
