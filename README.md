# time4116/iac-smith-showcase-infra infrastructure

Terraform/Terragrunt infrastructure generated by IaC Smith from GitHub issues. Every change arrives as a validated pull request; after merge, the apply workflow plans and applies behind a manual environment approval gate.

## Stack

* `vpc-foundation`

## Environments

* `non-prod`

## Warnings

* NAT gateway is not configured; workloads requiring outbound internet access must run in public subnets
* VPC Flow Logs are not enabled; network traffic visibility and troubleshooting capability are limited
* No egress filtering or network ACLs are configured beyond default VPC rules
