# time4116/iac-smith-showcase-infra infrastructure

Terraform/Terragrunt infrastructure generated by IaC Smith from GitHub issues. Every change arrives as a validated pull request; after merge, the apply workflow plans and applies behind a manual environment approval gate.

## Stack

* `ecs-fargate-nginx`

## Environments

* `non-prod`

## Warnings

* Public IP exposure creates internet-facing attack surface; consider restricting ALB security group to known sources
* No HTTPS listener configured; a certificate ARN is required to enable TLS for production-grade security
* Container image pulled from public Docker Hub without image scanning or vulnerability assessment
* No container logging to CloudWatch configured; audit trail and debugging visibility is absent
* No auto-scaling policy defined; service will not scale with demand
